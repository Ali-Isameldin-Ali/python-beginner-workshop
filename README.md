# Python Beginner Workshop - Week 1

## Short Description
A collection of Python programs written for Week 1 of the Python Beginner Workshop, demonstrating core programming fundamentals.

## What the programs do
This repository contains 7 days of Python tasks covering different concepts:
* **Day 1:** Setup and basic `print()` statements.
* **Day 2:** Variables, Data Types, and User Input (Personal Profile & Addition).
* **Day 3:** Arithmetic Operators (Basic Calculator & Minute Converter).
* **Day 4:** If/Else Conditions (Grade Classifier, Even/Odd Checker, and ATM Validator).
* **Day 5:** For & While Loops (Multiplication Table, Password Checker, and Limited Attempts).
* **Day 6:** Lists (List Operations, Indexing, Max/Min/Average Calculator).
* **Day 7:** Dictionaries (Student Profiles, Grades Lookup, and a Mini Contact Book).

## How to use the programs
1. Ensure you have Python installed.
2. Clone this repository or download the files.
3. Open your terminal or command prompt.
4. Run any of the files using the `python` command followed by the file name. For example:
   ```bash
   python day4.py
